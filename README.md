## 🧑‍💻 Contributing

### Prerequisites

- [Node.js](https://nodejs.org) 18+
- [pnpm](https://pnpm.io) 7+

### Installation

1. Clone the repo: `git clone https://github.com/dentolos19/misquoted.git`
2. Install dependencies: `pnpm install`
3. Start the app: `pnpm run dev`
